### bad1dea5.github.io

- [Jekyll](https://jekyllrb.com/docs/pages/ "Jekyll")
- [Font Awesome](https://fontawesome.com/ "Font Awesome")
- [Picnic CSS](https://picnicss.com/ "Picnic CSS")

#### Setup
1. [Install dependencies](https://jekyllrb.com/docs/installation/ubuntu/ "Install dependencies")
2. `bundler install`
3. `bundle exec jekyll serve`
4. Browse to [http://localhost:4000](http://localhost:4000 "http://localhost:4000")
